# Quelle

Eigenschaften

| Name              | Wert              | Benötigter Datentyp   |
| ----              | ----              | --------------------- |
| JS-Eigenschaft   | src    | URL           |
| Mögliche Eingaben | Jede mögliche URL | URL           |

## Beispiel
```
Setze die Quelle von x auf "https://example.com/bild.png"
```
