# Schriftart

Eigenschaften

| Name              | Wert              | Benötigter Datentyp   |
| ----              | ----              | --------------------- |
| CSS-Eigenschaft   | fontFamily    | CSSFont           |
| Mögliche Eingaben | Jede mögliche Schriftart (bspw. sans-serif) | CSSFont           |

## Beispiel
```
Setze die Schriftart von x auf sans-serif
```
